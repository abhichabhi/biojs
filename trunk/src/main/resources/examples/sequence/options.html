Options are the inicialization data for the component

Mandatory

sequence | string

Is the sequence data to be displayed

target | string

Identifier of the DIV tag where the component will be displayed.

Optional

numCols | int

Number of chars per row

numColsForSpace | int

Number of chars to insert a space char after

highlights | array

Multiple highlighted regions
Syntax: [ { start: <startVal1>, end: <endVal1>}, ..., { start: <startValN>, end: <endValN>} ] where, both parameters 'start' and 'end' determines each highlighting interval.
